# Java8-GerenciaDeEsportes

- Web app para gerencia de esportes, com o objetivo de propor plano de treino para clubes

# Stack

- MongoDB Atlas
- Front End: Vue.JS, Bootstrap
- Melhores práticas: Lombok, Design Patterns
